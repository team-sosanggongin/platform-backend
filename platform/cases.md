# 엔티티 도메인 관계

1. User의 전화번호는 유일값이다.
    - 소셜 로그인 경로가 다르더라도, 이미 User 정보가 존재한다면, 새로운 User를 테이블에 추가하지 않는다.
    - 소셜 로그인 시, 만약 User 테이블에 유저가 존재하지 않는 경우, 
    - 최초 로그인 시, 휴대전화 인증상태는 완료되지 않았다.
      - 휴대전화 인증에 성공하면, 휴대전화 인증 정보를 저장한다
    - 유저가 생성된 이후, 휴대전화 인증 로직이 실행된다. 이때, 인증정보를 담은 객체는 PhoneVerification이다
